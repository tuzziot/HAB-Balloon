01 — Proyecto

¿Qué es
HAB Balloon?

Un globo meteorológico instrumentado con sensores, radios LoRa y 4G, diseñado para registrar datos atmosféricos reales durante el ascenso y descenso a través de la troposfera y estratosfera.

El proyecto HAB Balloon (High Altitude Balloon) es una plataforma de investigación atmosférica construida desde cero. El sistema integra un nodo ESP32 con sensores de posición (u-blox M9N), entorno (BME280), dinámica (MPU6050), magnetómetro y radio LoRa SX1278 para comunicación de largo alcance.

El payload es lanzado desde Córdoba, Argentina atado a un globo meteorológico de 800–1500 g, capaz de alcanzar altitudes estratosféricas superiores a los 30 km antes de que el globo reviente y el payload descienda bajo un paracaídas hasta el punto de recuperación.

Todo el sistema fue diseñado con redundancia de comunicación: LoRa activo durante el vuelo y el módulo 4G (EG91) que se activa al aterrizar para transmitir la posición exacta de recuperación vía MQTT.

03 — Misión

Fases del
Vuelo

FASE // 01
Ascenso
LoRa activo
~5 m/s velocidad de ascenso. Telemetría continua. Construcción del perfil de viento por capas.
FASE // 02
Burst
Detección IMU
El globo revienta a ~30 km. El MPU6050 detecta la transición de fase por aceleración brusca.
FASE // 03
Descenso
LoRa + predicción
Paracaídas desplegado. La incertidumbre de predicción se reduce progresivamente.
FASE // 04
Tierra
4G + LoRa beacon
Al aterrizar se activa el 4G para reporte exacto. LoRa opera como beacon de corto alcance para recuperación.

06 — Software

Tactical
Dashboard

Centro de comando táctico en tiempo real. El servidor Flask recibe telemetría MQTT, ejecuta el predictor adaptativo y sirve el dashboard con polling cada 2 segundos.
